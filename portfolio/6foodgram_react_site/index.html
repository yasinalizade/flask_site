Diploma project - FOODgram

  • Category: Web-Application
  • Description: Web-app for recipes

Description

Web-app with cooking recipes. Anonymous user can get the recipes, also enabled filters on tags for better experience. If user is authenticated, he can get recipes, follow other authors, as well as like recipes, which are collecting on another tab "Favorite". Also, authenticated user can add recipes in shopping cart, and download all neccessary ingredients in PDF-format. Backend is linked with frontend via API (django rest framework). There are 4 containers for backend, frontend, nginx, postrgesql, which work together via docker-compose.

GitHub: https://github.com/yasinalizade/foodgram-project-react

Install

For install you need to download Docker and Docker-Compose

Clone repo and get in the directory 'infra'

git clone git@github.com:yasinalizade/foodgram-project-react.git

cd foodgram-project-react/infra

Launch Docker, migrate data and download ingredients

sudo docker-compose up -d

docker-compose exec web python manage.py migrate docker-compose exec web python manage.py load_ingredients docker-compose exec web python manage.py collectstatic --no-input

Finally, the application is able at: http://localhost/
