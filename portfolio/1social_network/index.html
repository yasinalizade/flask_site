YaTUBE

  • Category: Mvp-Network
  • Description: Social network for personal blogs, public discussion and commenting friends' posts
  • Info: http://iasinalizade.pythonanywhere.com

Description

You can sign up, log in or visit anonymously the website. As an author you can post text, image, or follow and comment other users' posts.

For better user experience enabled pagination!

Philosophy

Our power is our community, which is crazy about cats!

Meow - is the power! :)

Technological stack

  • Python 3.7
  • Django 2.2 LTS

Launch the project in dev-mode

Download the project and activate the virtual environment:

python3 -m venv venv

Download all necessary libraries from requirements.txt:

pip install -r requirements.txt

Check you are in the directory with manage.py and execute command:

python3 manage.py runserver
