HomeWork Bot

  • Category: Telegram Bot
  • Description: Telegram Bot for checking status of homeworks

Description

This project is designed to automatically notify me when the status of my homework changes. The program address requests to "Practicum.Domashka" API and, when the status changes, sends me a message in Telegram about the current status.

Tech stack

  • Python 3.8
  • python-telegram-bot

GitHub: https://github.com/yasinalizade/homework_bot
