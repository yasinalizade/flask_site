API for YaTUBE

  • Category: Api
  • Description: YaTUBE is a modern website, which needs API for the future development (e.g. iOS/Android app)

Description

API designed for YaTUBE network project. JWT-tokens used for authentication. Anonymous user can GET all posts, groups, comments. Registered user can POST posts, only authors can PATCH or DELETE their posts or comments.

Tech stack

  • Python 3.9
  • Django 2.2.16
  • Django Rest Framework 3.12.4
  • Simple-JWT 4.7.2

Launch the project in dev-mode

Download the project and activate the virtual environment:

python -m venv venv

source venv/bin/activate

Download all necessary libraries from requirements.txt:

python3 -m pip install --upgrade pip

pip install -r requirements.txt

Upgrade migrations:

python manage.py migrate

Launch the project:

python3 manage.py runserver

All accessible urls for API:

drf-yasg library is enabled in project for automatic update of all API documentation. Please, check:
