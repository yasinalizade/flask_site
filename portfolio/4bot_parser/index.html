Bot-parser

  • Category: Telegram Bot
  • Description: Telegram Bot for parsing Lidl.ie promo prices

Description

Program for parsing sale prices on the website - "Lidl.ie" and creating .csv file. After that it sends the file in Telegram-chat.

Install

Create a virtual environment:

python3 -m venv BOTenv

Start the virtual environment:

. BOTenv/bin/activate

Install necessary libraries:

pip install -r requirements.txt

Tech stack

  • Python 3.8
  • python-telegram-bot
  • asyncio

GitHub: https://github.com/yasinalizade/bot_parser
